# Music Album Application


## Software Versions
Tested on Mac OS X 10.15.6

Node.js v10.15.3
NPM 6.9.0
yarn 1.16.0
watchman 4.9.0
Xcode 11.6
Android Studio 4.0.1

### Initial Project Set Up
yarn

### Install pod on iOS project
npx pod-install

### RUN on iPhone
yarn ios

### RUN on Android
yarn android
